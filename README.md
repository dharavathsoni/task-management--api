# Task Management API

A RESTful API for managing tasks built with Python and SQLite.

## Features
- Create, read, update, and delete tasks
- Filter tasks by status and priority
- Task statistics and analytics
- SQLite database for persistence

## Technologies
- Python 3.x
- SQLite3
- Object-Oriented Programming

## Installation
```bash
git clone https://github.com/dharavathsoni/task-management-api.git
cd task-management-api
python task_api.py
```

## Usage
Run the main script:
```bash
python task_api.py
```
